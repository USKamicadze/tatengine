SDK dir structure

SDK dir structure

  • Engine - contains engine code and documentation
    • Code
      • Engine - contains engine code
      • Launcher - contains engine launcher code (system depended code)
    • Documentation - contains documentation
    • Templates - contains projects templates
  • Projects - contains projects
  • Tools
    • Bakery - data "cooking" tools, convert all data to raw blobs, etc
    • DistanceField - generate distance field fonts
    • FbxConverter - convert fbx to engine model format
    • PhotoshopExport - photoshop export plugin
    • Recompress - additional compress tools

Project dir structure

  • Bin - contains binary, dll, libs, etc
  • Code
    • Game - contains game code
    • SomeLib - contains lib code
    • Projects - contains projects for ide
      • MSVS - for msvs
      • Templates - contains projects templates
        • MSVS - template for msvs
        • XCode - template for xcode
      • XCode - for xcode
  • Content - contains source content
  • Media - contains cooked content
  • User - "user dir", emulate user dirs on mobile phones
